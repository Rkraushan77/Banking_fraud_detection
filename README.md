# 🏦 Banking Fraud Detection System

A machine learning-based web application that detects fraudulent banking transactions using a Random Forest Classifier and an interactive Streamlit dashboard.

## Features

* Fraud prediction in real time
* Interactive dashboard
* Correlation heatmap analysis
* Feature importance visualization
* Transaction risk assessment

## Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-Learn
* Matplotlib
* Seaborn

## Dataset Features

The model uses:

* Transaction Amount
* Anomaly Score
* Device Risk Score
* Transaction Velocity Score
* Suspicious IP Flag
* International Transaction Flag

## How It Works

1. Load the banking transaction dataset.
2. Train a Random Forest classification model.
3. Analyze transaction risk indicators.
4. Enter transaction details in the prediction form.
5. Receive a fraud prediction with probability score.

## Prediction Output

* ✅ Legitimate Transaction
* ⚠️ Fraudulent Transaction
